# Supply Chain Demand Analysis

An end-to-end demand forecasting and S&OP analysis pipeline built with Python.

## Project Overview
This project simulates real-world supply chain planning workflows using 
retail sales data, implementing ETL, demand analysis, and forecasting concepts 
used in enterprise planning platforms like o9 Solutions.

## Tech Stack
- Python (pandas, NumPy, matplotlib, seaborn, statsmodels)
- Jupyter Notebook
- GitHub

## Project Structure
- `data/` — Raw and cleaned datasets
- `notebooks/` — Analysis notebooks (one per day)
- `src/` — Python scripts
- `outputs/` — Charts and visualizations

## Concepts Covered
- ETL Pipeline (Extract, Transform, Load)
- Demand Management
- S&OP (Sales & Operations Planning)
- Time Series Analysis
- Demand Forecasting
